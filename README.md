<h1> TelecomX - Predicción de Cancelación de Clientes (Churn)</h1>

<div class="section">

<h2>📌 Descripción del Proyecto</h2>

<p>
Este proyecto tiene como objetivo analizar y predecir la cancelación de clientes 
(<b>Churn</b>) en una empresa de telecomunicaciones utilizando técnicas de 
<b>Machine Learning</b>.
</p>

<p>
Se aplicaron distintos modelos de clasificación para identificar qué clientes 
tienen mayor probabilidad de cancelar el servicio y analizar las variables que 
influyen en esta decisión.
</p>

</div>

<div class="section">

<h2> Modelos Utilizados</h2>

<ul>
<li>Regresión Logística</li>
<li>Random Forest</li>
</ul>

<p>
Los modelos fueron evaluados utilizando las siguientes métricas:
</p>

<ul>
<li>Accuracy</li>
<li>Precision</li>
<li>Recall</li>
<li>F1-Score</li>
<li>Matriz de Confusión</li>
</ul>

</div>

<div class="section">

<h2> Resultados del Análisis</h2>

<p>
El análisis permitió identificar los factores más relevantes en la cancelación 
de clientes, entre ellos:
</p>

<ul>
<li>Antigüedad del cliente (tenure)</li>
<li>Tipo de contrato</li>
<li>Cargos mensuales</li>
<li>Servicios adicionales</li>
</ul>

<p>
Los modelos de Machine Learning mostraron una capacidad aceptable para detectar 
clientes con riesgo de cancelación, lo cual puede ayudar a la empresa a diseñar 
estrategias de retención.
</p>

</div>
